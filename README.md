## 简介

顾名思义，`CodeStudy`为代码学习或代码研究的意思，本项目用于记录我在学习过程中的笔记，包括但不限于前端、后端、开发工具等。

严格来说这不是一个完整的开源项目，它仅仅用于在我需要用到以前写过的笔记时，用来查阅所需要的代码片段，但如果能帮助到跟我一样有类似需求的任何人，我感到很荣幸，可能这就是开源所带来的价值和意义吧。
